Detailed Description

The PhidgetSimpleServo class.

The PhidgetSimpleServo only supports movement by setting the position. The acceleration and velocity properties aren't implemented by the Phidget Servo module but they can be simulated by setting the position according to the acceleration/velocity set if needed.
