Creates a device from a QDomElement. This method should add the device of the robots DeviceManager. If the created Device is a Controller the controller property of the robot should also be set. Each device element may contain channel elements which have to be parsed by the channelFactories. Also if the device have need of a DeviceBoard it should be added to the DeviceManager too. See actuator/phidgetadvancedservofactory.cpp for an example.
